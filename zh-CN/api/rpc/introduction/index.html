NEAR RPC API

RPC API를 사용하면 NEAR 네트워크와 직접 통신할 수 있습니다. 예를 들어, near-api-js와 같은 도구는 RPC 호출을 만드는 추상화일 뿐입니다.

RPC 제공자​

선택할 수 있는 여러 RPC 공급자가 있습니다. 이러한 공급자는 NEAR 네트워크와 상호 작용하는 데 도움이 되는 중개자 역할을 합니다.

NEAR RPC - Quick Links​

API설명
Access Keys계정의 액세스 키에 대한 정보를 검색합니다.
Accounts / Contracts계정 및 컨트랙트에 대한 세부 정보를 보고 컨트랙트 호출을 수행합니다.
Block / Chunk네트워크를 쿼리하고 특정 블록 또는 청크에 대한 세부 정보를 얻습니다.
Gas특정 블록 또는 해시에 대한 가스 가격을 가져옵니다.
Protocol현재 제네시스 및 프로토콜 구성을 검색합니다.
Network노드 및 밸리데이터에 대한 상태 정보를 반환합니다.
Transactions트랜잭션을 보내고 상태를 쿼리합니다.
Maintenance Windows밸리데이터에 대한 현재 에포크의 유지 관리 기간을 쿼리합니다.
提示

Postman, JavaScript 및 HTTPie를 사용하여 JSON RPC 2.0 엔드포인트에 액세스할 수 있습니다.
